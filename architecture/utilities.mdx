## Overview

The `src/utils/` directory contains reusable utility functions, design tokens, and helper modules used throughout The Go Game Todo App.

## Alert Utility

**Location**: `src/utils/alert.ts`

Cross-platform alert function with web polyfill.

```ts src/utils/alert.ts
import { Alert, Platform } from 'react-native'

const alertPolyfill = (
    title: string,
    description: string,
    options?: Array<{
        text: string,
        onPress: () => void,
        style?: string
    }>
) => {
    if (!options) {
        window.alert([title, description].filter(Boolean).join('\n'));
        return;
    }
    
    const result = window.confirm([title, description].filter(Boolean).join('\n'))

    if (result) {
        const confirmOption = options.find(({ style }) => style !== 'cancel')
        confirmOption && confirmOption.onPress()
    } else {
        const cancelOption = options.find(({ style }) => style === 'cancel')
        cancelOption && cancelOption.onPress()
    }
}

const alert = Platform.OS === 'web' ? alertPolyfill : Alert.alert

export default alert;
```

### Parameters

<ParamField path="title" type="string" required>
  Alert dialog title
</ParamField>

<ParamField path="description" type="string" required>
  Alert message body
</ParamField>

<ParamField path="options" type="array">
  Array of button configurations with `text`, `onPress`, and optional `style` ('cancel')
</ParamField>

**Usage**:

```tsx
import alert from '@/utils/alert';

// Simple alert
alert("Success", "Task created successfully!");

// Alert with actions
alert(
  "Delete Task",
  "Are you sure you want to delete this task?",
  [
    { text: "Cancel", onPress: () => {}, style: "cancel" },
    { text: "Delete", onPress: () => deleteTask(id) }
  ]
);
```

<Note>
  The alert utility automatically uses native `Alert.alert` on iOS/Android and falls back to browser `alert()`/`confirm()` on web platforms.
</Note>

## Date Utility

**Location**: `src/utils/date.ts`

Date formatting helper function.

```ts src/utils/date.ts
export function getFormattedDate(date: Date) {
    return date?.toString()?.slice(0, 10);
}
```

### Function

<ParamField path="getFormattedDate" type="(date: Date) => string">
  Formats a Date object to a string in ISO format (YYYY-MM-DD)
</ParamField>

**Parameters**:
- `date` (Date): JavaScript Date object to format

**Returns**: String representation of the date (first 10 characters of `toString()`)

**Usage**:

```tsx
import { getFormattedDate } from '@/utils/date';

const formattedDate = getFormattedDate(new Date());
// Example output: "Wed Mar 03"
```

<Warning>
  This function uses `Date.toString().slice(0, 10)` which returns the day and date portion (e.g., "Wed Mar 03"), not a full ISO date string. For ISO format dates, use `date.toISOString().slice(0, 10)` instead.
</Warning>

## Summary

The utilities module provides:

- **GlobalColors** - Consistent color palette across the app
- **fontsList** & **appFonts** - Custom font loading and typography tokens
- **GlobalStyles** - Reusable component styles (card layout)
- **alert** - Cross-platform alert dialogs with web support
- **getFormattedDate** - Date formatting helper

These utilities ensure consistency, reduce code duplication, and provide platform-specific adaptations where needed.
